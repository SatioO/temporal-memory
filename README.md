GraphMind is a knowledge graph-powered RAG system for intelligent document retrieval.
Built with LangGraph and Neo4j for structured, context-aware reasoning over large datasets.
Supports multi-hop graph traversal to uncover hidden relationships between concepts.
Designed for teams who need explainable retrieval, not just vector similarity scores.
Features a plugin system with session hooks, context compression, and summarization pipelines for efficient long-context handling.
Includes an evaluation framework and Docker-based infrastructure for reproducible local development and testing.
